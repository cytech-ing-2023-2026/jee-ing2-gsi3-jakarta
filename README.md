## Running the project

Note: Java **21 or higher** is required. This project has been tested with Tomcat 10.1.33

Linux:

```shell
./mvnw war:war
```

Windows:

```shell
.\mvnw.cmd war:war
```

Then you can launch the WAR file using Tomcat.
